# Changelog - 作战地图

> 所有版本变更的详细记录

---

## [v2.3] - 2026-05-16

### Added
- IndexedDB 底图持久化功能
- 导入底图时自动保存到 IndexedDB
- 初始化时从 IndexedDB 恢复所有区域的底图

### Changed
- 清除底图时同步清除 IndexedDB
- 清除所有数据时同步清除 IndexedDB

---

## [v2.2] - 2026-05-16

### Added
- 新增数据展示按钮和弹窗功能
- 支持今日/本周/本月数据切换
- 新增学习时间曲线图
- 新增月度热力图
- 新增连续学习天数统计
- 新增日均学习时长对比图表
- 新增精力趋势图

### Changed
- 右上角新增"数据"按钮（位于"退出系统"旁边）

---

## [v2.1] - 2026-05-16

### Added
- 新增工作规则（修改前复述、版本备份）
- 建立 `backups/` 目录作为版本备份存储
- 新建 `README.md` 作为项目说明文档

### Changed
- 更新 `CLAUDE.md` 添加强制执行的工作规则

### Deleted
- 删除旧文档 `项目说明.txt`

### Infrastructure
- Git 标签 v2.1
- 成功推送到 GitHub

## [v2.0] - 2026-05-15

### Added
- 初始版本：作战地图完整功能
- 单文件 HTML 应用
- 六边形地图系统
- 军种管理（政治、英语、数学、专业课、健身、阅读）
- 双模式系统（用户/管理员）
- 战斗机制（3天超时）
- 数据持久化（localStorage + 备份导入导出）
- 多 A0 区域拼接支持
