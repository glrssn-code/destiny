# 作战地图 - 项目说明

> Last updated: 2026-05-16

## 项目概述

**作战地图**是一个单文件 HTML 学习管理应用，以文明游戏风格的地图占领与战斗机制追踪学习进度。

- **类型**: 单文件 HTML 应用，无需服务器，直接在浏览器运行
- **仓库**: https://github.com/glrssn-code/destiny.git

---

## 功能列表

### 1. 军种管理
- 左侧面板显示所有学习科目（军种）
- 支持自定义名称、颜色、备注
- 管理员模式可添加/编辑/删除军种

### 2. 六边形地图
- 中央 Canvas 绘制六边形网格
- 支持底图导入（jpg/png）
- 支持多 A0 区域拼接扩展
- 拖拽平移、滚轮缩放、右键调整俯仰角

### 3. 双模式系统
- **用户模式**: 记录学习时长、占领地块、进行战斗
- **管理员模式**: 配置军种属性、编辑地块属性、管理底图

### 4. 战斗机制
- 占领有属性颜色的地块触发战斗
- 3 天超时自动判定失败
- 战斗动画（呼吸闪烁效果）

### 5. 数据持久化
- localStorage 保存所有数据
- 支持完整备份导出/导入（JSON + Base64 底图）
- 操作日志与学习日志记录
